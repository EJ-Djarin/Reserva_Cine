Detailed Description

Clase que representa un cine con espacios numerados que pueden reservarse o liberarse.

Esta clase administra un conjunto de espacios (asientos) en un cine y permite mostrar su estado (reservado o libre) usando colores en consola. También permite reservar o cancelar una reserva en un espacio específico.
